# Screenshot Checklist

Place final presentation screenshots in this folder. Suggested capture order:

1. Streamlit Home: hero, KPI cards, pipeline cards, top project table.
2. API Status: token configured status without exposing values, source distribution, collection summary.
3. Trend Analysis: language chart, stars/forks chart, cluster/topic distribution, Top N table.
4. Scoring Ranking: filters, risk-potential quadrant, single project score card.
5. Cluster Map: PCA scatter, cluster model comparison table, cluster profile cards.
6. Model Results: Task A/Task B cards, model comparison, feature importance, training curves.
7. Personalized Recommendation: profile settings, personalized ranking, explanation card, Codex Prompt.
8. DeepSeek Agent: normalized Agent cards, CriticAgent review, ProjectAdvisorAgent recommendations.
9. HTML Report: hero and executive summary.
10. HTML Big Screen: KPI strip and chart grid.
11. README: project highlights and run commands.

No automated screenshot dependency is required for this project stage.
